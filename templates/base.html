PORTFOLIO PROJECT

A RESTful video game API featuring full CRUD operations, filtering and sorting, derived fields, and a review system. Includes JWT authentication with refresh token rotation, role-based authorization, and context-specific policies. Built with structured error handling, logging, and thoroughly tested with 148+ tests. Fully documented via Swagger UI.

ROUTES:

Full Swagger documentation: /api/docs

all games: /games

all genres: /genres

all developers: /developers

all reviews (per game): /games/game_id/reviews

a game: /games/game_id

a genre: /genres/genre_id

a developer: /developers/developer_id

a review: game/game_id/reviews/review_id

self: /me
